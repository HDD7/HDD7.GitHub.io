建站日志

在GoDaddy上买了这个域名hdd666.me，过了大半年也没有使用，毕竟懒，整个博客出来也是麻烦了自己，还是那句话，买都买了，那就用了吧。

先聊聊之前要建站的三个方案吧。以下还有复制来的介绍

  1. WordPress

  2. Hexo

  3. Typecho

WordPress:

WordPress是使用PHP语言开发的博客平台，用户可以在支持PHP和MySQL数据库的服务器上架设属于自己的网站。也可以把 WordPress当作一个内容管理系统（CMS）来使用。WordPress是一款个人博客系统，并逐步演化成一款内容管理系统软件，它是使用PHP语言和MySQL数据库开发的。用户可以在支持 PHP 和 MySQL数据库的服务器上使用自己的博客

Hexo:

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。一般选择结合github使用，无需支付服务器成本

Typecho:

Typecho是由type和echo两个词合成的，来自于开发团队的头脑风暴。Typecho基于PHP5开发，支持多种数据库，是一款内核强健﹑扩展方便﹑体验友好﹑运行流畅的轻量级开源博客程序。选用Typecho，搭建独一无二个人网络日志发布平台，享受创作的快乐。

基于本人的需要，大半年发不了几篇文章，也懒得折腾。此时静态博客Hexo便是一个不错的选择，并且结合Github Page使用，也省下了一笔费用。Github在天朝也早已解封，直连状态也不错，对于静态网页够用了。平时用git来push也挺方便，于是就这样开始建站了。

放点教程

要搭建Hexo博客，需要申请自己的GitHub Page，并且需要安装Node.js（Hexo基于此）和Git。

两个都是常用的工具，这学期学习JavaScript，顺便连带Node.js学了。
